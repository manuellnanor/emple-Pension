# emPLE Pension Trust

Responsive website for emPLE Pension Trust Ghana, built with Next.js, React, TypeScript, and Tailwind CSS.

## Run Locally

Prerequisite: Node.js 20.9 or newer.

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Validation

```bash
npm run typecheck
npm run build
```
